Astro works very well for blogs because it ships static HTML by default and lets you add JavaScript only where it really makes sense.

## Why use Content Collections

Content Collections give your post frontmatter a contract. That helps prevent publishing content without a title, description, slug, or valid date.

## When to use MDX

Use MDX when an article needs interactive components or richer visual blocks. For simple writing, Markdown is still enough.
