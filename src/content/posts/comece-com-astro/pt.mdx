Astro funciona muito bem para blogs porque entrega HTML estático por padrão e permite adicionar JavaScript apenas onde ele realmente faz sentido.

## Por que usar Content Collections

Content Collections dão contrato para o frontmatter dos posts. Isso evita publicar conteúdo sem título, descrição, slug ou data válida.

## Quando usar MDX

Use MDX quando o artigo precisar de componentes interativos ou blocos visuais mais ricos. Para textos simples, Markdown continua sendo suficiente.
